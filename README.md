SQLB - SQL Batch Tool
====

## Usage
SQLB [FilePath] [ConStr]

    > sqlb sql\*sql usr/pwd@srv 

Output Log File

    >type log\yyyymmdd_hhmmss.log
    [usr@srv]
    C:\Users\nmrmsys\Desktop\sqlb\sql\test.sql 
    
    ----- sql output -----
    
    START:16:53:12.59 END:16:53:13.00 ELAPSED:00:00:01 

## Requirement
- Oracle: sqlplus 
- SQL Server: sqlcmd
